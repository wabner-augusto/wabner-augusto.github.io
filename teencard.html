Visão Geral

Sobre o projeto

Problema: Por ser mais um produto pré-pago em um mercado saturado, a Agillitas estava com dificuldades em descatar os diferencias do produto e em se comunicar com o público correto.

Solução: Desenvolver um canal de comunicação onde fosse possível o usuário ter informações sobre todos os benefícios e vantagens. Desenvolvemos uma nova UI para o site.

Tempo: 2 Semanas

Minha Função: Como designer fui responsável pela UI da landingpage e organização do conteúdo.

O Desafio

Desenvolver uma nova UI para o site onde o conteúdo fosse prático e informativo. Que a nova UI conversasse com o público correto. Que o canal de comunicação trouxesse confiança para os pais adquirirem este produto para seus filhos.
